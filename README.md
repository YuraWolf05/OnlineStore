# OnlineStore

Console application for managing products and orders in an online store.

---

## Technologies

- C#
- .NET
- xUnit
- GitHub Actions

---

## Architecture

Project structure:

- Domain
- Application
- Infrastructure
- Console UI

---

## Features

- Add products
- View products
- Shopping cart
- Order creation

---

## Run application

```bash
dotnet run --project src/OnlineStore.Console
```

---

## Run tests

```bash
dotnet test
```
